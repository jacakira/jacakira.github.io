Quotients, Intuitively

February 26, 2025

#algebra 路 #set-theory

Quotients are one of those ideas that appear everywhere. At a very crude level, a quotient is what you get when you declare some points to be the same.

Formally, start with a set $X$ and an equivalence relation $\sim$ on $X$. The quotient $X / \sim$ is the set of equivalence classes $[x]$, where

$$ [x] = { y \in X : y \sim x }. $$

The canonical surjection

$$ \pi : X \to X / \sim, \quad x \mapsto [x] $$

is universal among maps that respect the relation $\sim$ in a suitable sense.

We can see the same pattern in:

  • Modular arithmetic: $\mathbb{Z} / n\mathbb{Z}$
  • Topology: gluing edges of a square to obtain a torus
  • Linear algebra: quotient spaces $V / W$

A useful mental model: a quotient is a controlled loss of information; we crush certain distinctions while trying to keep as much structure as possible.
